# Pricing Page

Built to match **PRDs/pricing** (reference implementation). Order: PricingHero → PricingTiers → FeatureComparison → FAQ → BottomCTA (+ SubscribeForm).

- **PricingTiers**: Three cards — Free (all features), Pro (incremental features only vs Free), Enterprise (consultative text only, no feature list; Custom pricing, Contact Us CTA). Pro card highlighted with gradient border.
- **FeatureComparison**: “Compare plans in detail” with Free and Pro columns only (no Enterprise); categories Content & Branding, Engagement, Analytics, Support & Admin; desktop table + mobile cards.
- **FAQ**: PRD copy; “paid Pro and custom Enterprise options”; “Pro and higher” for sponsors.
- Colors from PRDs/pricing/src/index.css (navy #1B2A4A, blue #3B82F6, emerald, violet, amber).
