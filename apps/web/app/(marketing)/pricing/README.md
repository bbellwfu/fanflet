# Pricing Page

Built to match **PRDs/pricing** (reference implementation). Order: PricingHero → PricingTiers → FeatureComparison → FAQ → BottomCTA (+ SubscribeForm).

- **PricingTiers**: Monthly/Annual toggle (Save 12%), Free (Current Plan), Pro (Coming Soon, $12/mo), Enterprise (Custom). Pro card highlighted with gradient border.
- **FeatureComparison**: “Compare plans in detail” with categories Content & Branding, Engagement, Analytics, Support & Admin; desktop table + mobile cards.
- **FAQ**: PRD copy; “Pro and Enterprise tiers”, “On Enterprise plans” for sponsors.
- Colors from PRDs/pricing/src/index.css (navy #1B2A4A, blue #3B82F6, emerald, violet, amber).
